PRODUCT REQUIREMENTS:
- **Shopify Integration**: Sync orders, customers, fulfillment status, order number, and payment status from multiple Shopify stores, starting from 2015.
- **Advanced Inventory Management**: Upload opening stock via Excel and automatically deduct stock only when an order's delivery status becomes DELIVERED. Restore stock for returned orders.
- **Multi-layered Tracking System**:
    - **Confirmation Tracker**: A pre-dispatch system for agents to call customers and confirm unfulfilled orders.
    - **Dispatch Tracker**: A post-dispatch system to track fulfilled orders shipped via TCS, monitoring delivery status and COD payment status.
    - **Purchase Tracker**: A system to track orders sourced from China (identified by tracking numbers starting with 'X').
- **TCS Courier Integration**: Integrate with TCS to get real-time delivery status and COD (Cash on Delivery) payment status.
- **CSV Import**: Allow the user to upload a Shopify orders CSV to merge and backfill any orders missing from the API sync.
- **UI/UX**: A modern, clean UI with clear sidebar navigation, robust filtering/sorting, and reliable pagination.
