## ⚠️ IMPORTANT NOTES

### About "Platform Data":
Platform Data from Meta includes:
- WhatsApp customer phone numbers
- Customer names from WhatsApp profiles
- Message content
- Message timestamps
- Delivery status

### What NOT to include:
Don't list processors that only handle:
- Order data from Shopify (not WhatsApp data)
- Tracking info to DTDC (outbound only, not processing Meta data)
- Payment processors (if they don't receive WhatsApp data)

### Data Security:
All processors you list should:
- Have secure data storage
- Use encryption
- Have proper access controls
- Comply with data protection standards
